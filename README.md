# SimFishers X
上次适配：`2026/04/30 1.21.8`

---
一个自动处理自己想的钓鱼qte的mod。运行于`Fabric 1.21.8`。

本版本同捆了一个很弱智的 Anti-AFK（仅在钓鱼竿在主副手时启用）功能。

本模组是我自己在某天白日梦时，想到了一个钓鱼逻辑，这个逻辑显然比星露谷的要简单一些。因此为了和我自己左手扳右手，我写了这个自动处理我想到的这个钓鱼逻辑的mod。

**作者不知情、不建议、不支持本mod在任何服务器上的使用。如果您要在任何服务器上使用，使用前请咨询管理员，提前了解你的行为是否属于作弊！**
